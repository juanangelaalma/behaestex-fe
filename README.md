## How to Run
### 1. Clone this repo
```bash
git clone https://github.com/juanangelaalma/behaestex-fe.git
```
### 2. Change dir to behaestex-fe:
```
cd behaestex-fe
```
### 3. Install packages
```bash
npm install
```
### 4. Run server
```bash
npm run start
```

Note: remember to configure config/*.js files
